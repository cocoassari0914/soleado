# Install

1. Download from https://github.com/cocoassari0914/soleado
1. Move to `redmine/public/theme/soleado`

# git
```
cd redmineのインストールディレクトリ/public/theme
git clone https://github.com/cocoassari0914/soleado.git
```

# redmineのテーマの変更方法
